## Overview

In-depth EDA on housing data covering outlier detection, correlation analysis, and geographic visualisation.

## Tech Stack

- Python
- Pandas
- Seaborn
- Matplotlib
- NumPy

## Getting Started

```bash
# Clone the repo
git clone https://github.com/prajwal467262/house-price-eda-&-feature-analysis

# Install dependencies
pip install -r requirements.txt

# (If applicable) Generate sample data first
python generate_sample_data.py

# Run the main script
python house_price_eda.py
```
